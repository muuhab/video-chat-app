## Installation
First the server side
```sh
npm i
npm run start
```
Second the client side
```sh
cd cient
npm i
npm run start
```
>Server running  on : - 
```sh
127.0.0.1:5000
```
>Client side running  on : - 
```sh
127.0.0.1:3000
```
